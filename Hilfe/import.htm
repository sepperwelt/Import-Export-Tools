Importfunktion

Um ein früher erstelltes Backup einer Datenbank wiederherzustellen, geben Sie bitte die Pfade der Backups an.
Dabei überschreibt das Programm alle Daten in einer vorhandenen Datenbank mit gleichen Namen. Die Datenbanknamen können in den Einstellungen geändert werden.

Versuchen Sie niemals, eine andere SQL-Datei zu importieren!
Dadurch fehlen DigiKam wichtige Arbeitsdaten, die zu unerwünschtem Fehlverhalten von DigiKam führen können!
